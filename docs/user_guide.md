# 班级考勤系统使用指南

## 一、系统概述

本系统是基于 B/S 架构的班级考勤系统，核心功能包括：
- **人脸考勤**：实时人脸检测与活体检测
- **合照识别**：批量识别合照中的学生
- **情绪分析**：分析学生面部情绪
- **数据管理**：学生信息、考勤记录管理

---

## 二、系统环境要求

### 前端运行环境
- 浏览器：Chrome 90+、Firefox 88+、Edge 90+
- 需要摄像头权限
- 网络连接正常

### 后端运行环境
- Python 3.10+
- MySQL 8.0+
- 已安装相关依赖包

---

## 三、登录操作

### 3.1 访问系统
- **操作**：打开浏览器，访问 `http://localhost:5173/login.html`
- **预期结果**：显示登录页面，包含用户名和密码输入框

### 3.2 用户登录
- **操作**：
  1. 在用户名输入框中输入用户名
  2. 在密码输入框中输入密码
  3. 点击「登录」按钮
  
- **测试账号**：
  | 角色 | 用户名 | 密码 |
  |------|--------|------|
  | 管理员 | admin | demo_hash_123456 |
  | 教师 | teacher01 | demo_hash_123456 |
  | 学生 | 20240001 | demo_hash_123456 |

- **预期结果**：
  - 成功：跳转到系统首页，显示导航菜单
  - 失败：显示错误提示「用户名或密码错误」

## 四、考勤功能

### 4.1 进入考勤页面
- **操作**：点击左侧导航栏「考勤打卡」菜单
- **预期结果**：显示考勤摄像头界面

### 4.2 开启摄像头
- **操作**：点击「开启摄像头」按钮
- **预期结果**：
  - 浏览器弹出权限请求
  - 点击允许后，摄像头预览画面显示在页面中

### 4.3 人脸考勤打卡
- **操作**：
  1. 确保面部在摄像头画面中
  2. 点击「拍照考勤」按钮
  3. 等待系统识别

- **预期结果**：
  - **成功**：显示「考勤成功」提示，显示识别到的学生信息和情绪
  - **活体检测失败**：显示「检测到非活体，请使用真人进行考勤」
  - **人脸不匹配**：显示「未识别到匹配的学生信息」
  - **超时**：显示「识别超时，请重试」

### 4.4 查看考勤记录
- **操作**：
  1. 在考勤页面中点击「考勤记录」标签
  2. 可按日期、班级筛选

- **预期结果**：显示考勤记录列表，包含：
  - 学生姓名、学号、班级
  - 考勤时间
  - 考勤状态（成功/失败）
  - 情绪分析结果

---

## 五、合照识别功能

### 5.1 进入合照识别页面
- **操作**：点击左侧导航栏「合照识别」菜单
- **预期结果**：显示合照上传界面

### 5.2 上传合照
- **操作**：
  1. 点击「选择文件」按钮
  2. 选择本地合照图片（支持 jpg、jpeg、png 格式）
  3. 点击「开始识别」按钮

- **预期结果**：
  - 图片成功上传并显示预览
  - 系统开始检测图片中的人脸

### 5.3 查看识别结果
- **预期结果**：
  - 图片上标记出检测到的人脸位置（蓝色框）
  - 显示识别到的学生列表：学号、姓名、班级、匹配置信度
  - 显示未识别的人脸数量
  - 每个识别到的人脸附带情绪分析结果

---

## 六、情绪分析功能

### 6.1 进入情绪分析页面
- **操作**：点击左侧导航栏「情绪分析」菜单
- **预期结果**：显示情绪统计图表页面

### 6.2 查看情绪统计
- **操作**：
  1. 选择日期范围
  2. 选择班级
  3. 点击「查询」按钮

- **预期结果**：
  - 显示情绪分布饼图（中性、开心、疲惫、惊讶、悲伤、生气）
  - 显示情绪趋势柱状图
  - 显示各情绪占比数据

---

## 七、学生管理功能

### 7.1 进入学生管理页面
- **操作**：点击左侧导航栏「学生管理」菜单（仅限管理员和教师）
- **预期结果**：显示学生列表页面

### 7.2 查看学生列表
- **操作**：可按班级筛选学生
- **预期结果**：显示学生列表，包含学号、姓名、班级、状态

### 7.3 添加学生
- **操作**：
  1. 点击「添加学生」按钮
  2. 填写学生信息（学号、姓名、班级）
  3. 点击「保存」

- **预期结果**：学生信息添加成功，列表中显示新学生

### 7.4 批量导入学生
- **操作**：
  1. 点击「批量导入」按钮
  2. 选择 Excel 文件
  3. 点击「导入」

- **预期结果**：显示导入结果，成功导入的学生数量

## 八、预期错误提示与处理

| 错误提示 | 可能原因 | 处理方式 |
|----------|----------|----------|
| 请求的资源不存在 | API 路径错误 | 检查后端服务是否启动，确认 API 路径配置正确 |
| 未授权访问 | 登录状态过期 | 重新登录系统 |
| 摄像头无法开启 | 权限未授予或设备不可用 | 在浏览器设置中允许摄像头权限，检查设备连接 |
| 识别超时 | 网络延迟或人脸不清晰 | 确保光线充足，重试识别 |
| 活体检测失败 | 使用照片或视频 | 使用真人进行考勤 |
| 人脸不匹配 | 未录入人脸特征或人脸变化较大 | 联系管理员重新录入人脸信息 |

- **操作**：点击右上角用户头像，选择「退出登录」
- **预期结果**：跳转到登录页面，清除登录状态
